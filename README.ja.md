# Unix Timestamp Hover

Unix タイムスタンプにマウスオーバーすると、人間が読める形式の日時をツールチップで表示する VS Code 拡張機能です。

[English README](README.md)

## 機能

- **ホバーで変換**: Unix タイムスタンプにマウスカーソルを合わせるだけで、変換された日時が表示されます。
- **複数のフォーマットに対応**: 以下を自動的に検出します。
  - 秒 (10桁)
  - ミリ秒 (13桁)
  - マイクロ秒 (16桁)
  - ナノ秒 (19桁)
- **タイムゾーンのカスタマイズ**: お好みのタイムゾーンを設定できます（デフォルト: `Asia/Tokyo`）。
- **日時フォーマットのカスタマイズ**: 日時の表示形式を定義できます（デフォルト: `YYYY/MM/DD HH:mm:ss`）。

## 拡張機能の設定

この拡張機能は以下の設定を提供します：

* `timestampHover.timezone`: 日時の表示に使用するタイムゾーン（例: `Asia/Tokyo`, `UTC`, `America/New_York`）。デフォルトは `Asia/Tokyo` です。
* `timestampHover.format`: 日時のフォーマット文字列。
    * 利用可能なプレースホルダー:
        * `YYYY`: 年
        * `MM`: 月
        * `DD`: 日
        * `HH`: 時
        * `mm`: 分
        * `ss`: 秒
        * `SSS`: ミリ秒
    * デフォルトは `YYYY/MM/DD HH:mm:ss` です。

## 使い方

1. Unix タイムスタンプが含まれているファイルを開きます。
2. `1700000000` や `1700000000000` のような数字にマウスを合わせます。
3. フォーマットされた日時を示すツールチップが表示されます。

## 既知の問題

- 特定の桁数（10, 13, 16, 19桁）に一致する数字列のみをサポートしています。
- 誤検知を避けるため、1970年から2100年の間の年のみを有効としています。

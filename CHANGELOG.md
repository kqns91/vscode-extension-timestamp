# Change Log

All notable changes to the "vscode-extension-timestamp" extension will be documented in this file.

## [0.2.3] - 2023-12-06

- Retry automated publishing configuration.

## [0.2.1] - 2023-11-21

- Lower minimum VS Code version requirement to 1.80.0 for better compatibility (e.g. Cursor).

## [0.2.0] - 2023-11-21

- Initial release
- Support for converting Unix timestamps (seconds, ms, us, ns) to human-readable dates on hover.
- Added `timestampHover.timezone` setting.
- Added `timestampHover.format` setting.
